exports(d, e, f, a, b, c)

a set of parameters representing proportions for a Struct uses a generalized form that supports both Line and Circle structs `d x^2 + e x y + f y^2 + a x + b y = c` - ensure that the sign of c is passed in correctly - can support other conic 2d forms in the future the parameters are set as strings representing algebraic values
these parameters are held as properties of an object so that the properties can be iterated through for analysis **Algebrite** library http://algebrite.org provides all the calculations of the algebraic values and conversion to float when necessary - TODO: Equation should be treated as a static immutable object - TODO: all values that don't change should be calculated on construction.

Constructor

new exports(d, e, f, a, b, c)

Parameters:
Name Type Description
d string coefficient for x^2 term - always 1 for a circle - 0 for line
e string coefficient for xy term - should always be zero in this model
f string coefficient for y^2 term - always 1 for a circle - 0 for line
a string coefficient for x term
b string coefficient for y term
c string constant - WATCH SIGN - note that we position the c value on the other side of the equations
